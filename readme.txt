=== Description ===

The plugin displays the posts related to a specific post by category or post type.

Features include:
* It can be controlled by admin
* It can be controlled through the settings attached to this plugin
* Built-in WordPress locale codes
* display the related post attractively
* Allows the administrator displays the posts related to a specific post by category or post type 
* Allows the administrator to limit the number of related posts to display
* Allows the administrator to specify how to display the display related posts (ascending or descending)

= Installing manually: =

1. Unzip all files to the `wp-content/plugins/related-post` directory
2. Log into WordPress admin and activate the 'Amazing Related Post' plugin through the 'Plugins' menu
3. Go to *Amazing Related Post > Home* in the left-hand menu to start it
